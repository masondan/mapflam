# MapFlam

Mobile-first web app for journalists to create locator maps. Search locations, place markers, and export as PNG/JPG.

## Quick Start

```bash
npm install
npm run dev
```

Visit http://localhost:5173

## Build

```bash
npm run build
```

## Docs

See `AGENTS.md` for technical structure, roadmap, and API details.

## Stack

- SvelteKit / Vite
- Leaflet (maps)
- Nominatim (geocoding)
- html2canvas (export)
- Cloudflare Pages (hosting)
